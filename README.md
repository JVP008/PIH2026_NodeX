## The Problem We Are Solving

India has a massive home services market, but it is highly fragmented, unorganized, and often lacks trust. Homeowners frequently struggle to find reliable professionals, and at the same time, skilled workers don't have a solid platform to grow their businesses.

## Our Solution

HouseConnect Pro brings transparency, speed, and real trust to home services. Here is what we offer:

- **Smart Discovery:** Easily browse verified professionals by their category, rating, location, and availability.
- **Instant Booking:** Post your job, get matched, book the professional, and securely manage everything in one smooth flow.
- **Trust Layer:** We rely on verified profiles, ratings, genuine reviews, and a fair dispute resolution system to keep everyone safe.

## Getting Started Locally

If you want to run this project on your own machine, just follow these simple steps:

```bash
# install the required dependencies
npm install

# set up your environment variables
cp .env.example .env.local
# inside .env.local, add your NEXT_PUBLIC_SUPABASE_URL and NEXT_PUBLIC_SUPABASE_ANON_KEY

# start the development server
npm run dev
```

Then, open [http://localhost:3000](http://localhost:3000) in your browser.
